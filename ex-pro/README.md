# Excel Data Separator 📊

เว็บแอปพลิเคชันสำหรับเปรียบเทียบและคัดแยกข้อมูลจากไฟล์ Excel 2 ไฟล์ (ไฟล์ต้นฉบับ และ ไฟล์ที่นำมาเทียบ) โดยเมื่อเจอข้อมูลที่ต้องการ จะแยกไฟล์ออกมาเป็นไฟล์ที่ข้อมูลตรงกัน (Matched) และไฟล์ที่ข้อมูลไม่มีในไฟล์ต้นฉบับ (Unmatched) ทำงานบนเบราว์เซอร์ 100% ไม่พึ่งพาเซิร์ฟเวอร์

![Screenshot](image/Pasted%20image.png)

## ฟีเจอร์หลัก (Features)
- **ประมวลผลผ่านเบราว์เซอร์**: อ่านและสร้างไฟล์ `.xlsx` ได้ลื่นไหลผ่านไลบรารี SheetJS 
- **หน้าตา UI โมเดิร์น**: ดีไซน์แบบ Glassmorphism สวยงาม พรีเมียม ใช้งานง่าย
- **กำหนดคอลัมน์คัดแยกได้เอง**: โดยระบบจะดึงหัวคอลัมน์จากไฟล์ที่คุณอัปโหลดมาเป็นตัวเลือกให้คุณทันที
- **แยกไฟล์ให้พร้อมใช้**: จัดเตรียมไฟล์ข้อมูลที่ "ตรงกัน" และ "ไม่ตรงกัน" ให้คุณดาวน์โหลดแยกกันได้อย่างรวดเร็ว

## คู่มือการใช้งาน (How to use)

1. เปิดไฟล์ `index.html` ขึ้นมาบนเบราว์เซอร์คอมพิวเตอร์ของคุณ
2. อัปโหลดไฟล์ที่ต้องการ (โดยการคลิกที่กล่อง หรือลากไฟล์มาวาง)
   - กล่องซ้าย: **ไฟล์ต้นฉบับ (Master File)**
   - กล่องขวา: **ไฟล์ที่นำมาเทียบ (Compare File)** 
3. จะปรากฏแถบดรอปดาวน์ด้านล่าง ให้คุณเลือกชื่อคอลัมน์ในไฟล์นำมาเทียบ ที่ต้องการใช้เปรียบเทียบ
4. กดปุ่ม `กดยืนยัน (Process)` เพื่อให้โปรแกรมประมวลผล
5. กดดาวน์โหลดไฟล์ข้อมูลแยกตามที่คุณต้องการ

## เทคโนโลยี (Technologies)
- HTML5 / CSS3 / JavaScript
- [SheetJS (xlsx)](https://docs.sheetjs.com/) - เครื่องมือสำหรับอ่านและเขียนไฟล์ Excel ขั้นสูงบนเบราว์เซอร์
