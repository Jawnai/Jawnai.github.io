# 📦 คู่มือการใช้งาน NanoPack (NanoPack User Guide)

**NanoPack** เป็นเครื่องมือจัดการไฟล์ผ่านหน้าเว็บที่เน้นความเร็ว ความเรียบง่าย และความปลอดภัยสูงสุด โดยไฟล์ทั้งหมดจะถูกจัดการภายในเบราว์เซอร์ของคุณเท่านั้น (ไม่มีการอัปโหลดไฟล์ไปยังเซิร์ฟเวอร์)

---

## 🚀 ฟีเจอร์หลัก
- **Drag & Drop:** ลากไฟล์มาวางเพื่อเพิ่มเข้าระบบได้ทันที
- **Fast Processing:** ประมวลผลรวดเร็วด้วยเทคโนโลยีล่าสุด
- **Privacy First:** ไฟล์ของคุณจะไม่ถูกส่งออกไปไหน ปลอดภัย 100%
- **Bulk Download:** ดาวน์โหลดไฟล์ทั้งหมดที่เลือกไว้ในคลิกเดียว

---

## 📖 วิธีการใช้งาน (Step-by-Step)

### 1. การเพิ่มไฟล์เข้าระบบ
คุณสามารถเพิ่มไฟล์ได้ 2 วิธี:
- **วิธีที่ 1:** ลากไฟล์จากคอมพิวเตอร์ของคุณมาวางที่พื้นที่ **"Drop files here"**
- **วิธีที่ 2:** กดปุ่ม **"Choose Files"** เพื่อเลือกไฟล์จากโฟลเดอร์ในเครื่อง

### 2. การจัดการรายการไฟล์
เมื่อคุณเลือกไฟล์แล้ว รายการไฟล์จะปรากฏขึ้นมาด้านล่าง:
- **ดูข้อมูลไฟล์:** จะเห็นชื่อไฟล์ นามสกุล และขนาดไฟล์
- **ลบไฟล์:** หากไม่ต้องการไฟล์ไหน ให้กดปุ่ม **กากบาท (X)** หลังไฟล์นั้น
- **ล้างรายการทั้งหมด:** กดปุ่ม **"Clear All"** ด้านบนขวาของรายการไฟล์

### 3. การดาวน์โหลดไฟล์
- **ดาวน์โหลดทีละไฟล์:** กดปุ่ม **ไอคอนดาวน์โหลด** หลังชื่อไฟล์ที่ต้องการ
- **ดาวน์โหลดทั้งหมด:** กดปุ่ม **"Download All Files"** ด้านล่างสุด ระบบจะทำการรวบรวมและดาวน์โหลดให้คุณโดยอัตโนมัติ

---

## 🛠 ข้อมูลทางเทคนิคและความปลอดภัย
- **ไม่ต้องติดตั้ง:** ใช้งานผ่าน Chrome, Safari, Edge หรือเบราว์เซอร์สมัยใหม่ได้ทันที
- **Local Processing:** การจัดการไฟล์เกิดขึ้นที่เครื่องของคุณ (Client-side) ทำให้ทำงานได้แม้เน็ตช้าและปลอดภัยจากสายตาผู้อื่น
- **System Hub:** คุณสามารถสลับไปใช้งานเครื่องมืออื่นๆ เช่น PDF Studio หรือ Ex-Pro ได้ผ่านเมนูที่มุมขวาบน

---

## 💡 คำแนะนำเพิ่มเติม
- หากดาวน์โหลดไฟล์จำนวนมากพร้อมกัน เบราว์เซอร์อาจจะถามเพื่อขออนุญาต **"Allow multiple downloads"** ให้กด **Allow (อนุญาต)** เพื่อให้ระบบทำงานได้ครบถ้วน
- หากต้องการเริ่มต้นใหม่ทั้งหมด เพียงแค่กด Refresh หน้าเว็บ หรือกดปุ่ม Clear All ครับ
